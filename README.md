# Python-master-100d

A starter repository for Python learning projects, exercises, and practice files.

## Overview

This repository is intended to hold Python code, study notes, and solutions for a 100-day Python learning plan. It can be used to organize small scripts, modules, exercises, and examples as the project grows.

## Suggested Structure

- `day01/`, `day02/`, ... — daily practice exercises
- `projects/` — larger Python projects and demos
- `notes/` — study notes, tips, and references
- `README.md` — repository overview and instructions

## Getting Started

1. Clone this repository:

```bash
git clone https://github.com/zhanglkx/Python-master-100d.git
cd Python-master-100d
```

2. Create a Python virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

3. Add your Python files and exercises.
